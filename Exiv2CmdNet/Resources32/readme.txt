Issues and Solutions
=============================
Issues 1
- If found exception/error "The program can't start because MSVCP140.dll is missing from your computer. Try reinstalling the program to fix this problem."

Solution
- Because you haven't install Microsoft Visual C++ 2015 Runtime Library x86 and - Microsoft Visual C++ 2015 Runtime Library x64
- Just download and install it.
- more info: 
- http://www.thewindowsclub.com/the-program-cant-start-because-msvcp140-dll-in-missing
